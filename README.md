# employee_list
Simples sistema desenvolvido com NuxtJs e Tawilwind. 
Listagem de funcionários, pesquisa dinâmica, paginação e tela de detalhes.

Para instalar as dependências do projeto, siga os seguintes passos:

- Clone o repositório
- Navegue até ele
- Instale a dependências com `npm install`
- Inicie o projeto com `npm run dev`
